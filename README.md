## 1. Project Overview
**Design for All** is a responsive web project that explores the boundaries of web typography and interaction without the reliance on JavaScript. It is a manifesto of inclusivity, open-source thinking, and cross-media adaptability.

Built strictly with **HTML & CSS**, this project demonstrates how fundamental web technologies can be pushed to create immersive, animated, and functional user experiences.

## 4. Technical Implementation
This project was built under strict technical constraints to challenge the limits of modern CSS.

### The Constraints
* **No JavaScript** allowed.
* **HTML & CSS** Only.
* Maximum of **one** external image file.
* Inline SVGs.

### Key Features & Techniques
* **Responsive Layout:** Built with a robust combination of **CSS Grid** and **Flexbox**.
* **Scroll-Driven Animations:** Implemented purely with CSS to create dynamic storytelling as the user scrolls.
* **CSS-Only Interactivity:** Dropdown menus and toggles achieved via the `label` + `input` (Checkbox/Radio) hack.
* **Advanced Typography:** Utilized the `text-box` (text-box-trim) property for precise vertical alignment.
* **Scalable Graphics:** Extensive use of CSS-controlled Inline SVGs for lightweight, crisp visuals.
